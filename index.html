# Politique de Confidentialité - PBO Manager *Dernière mise à jour : 25 Mai 2025* ## Introduction Bienvenue dans PBO Manager. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile. ## Informations que nous collectons ### 1. Données de localisation - **Ce que nous collectons** : Votre position GPS actuelle - **Pourquoi** : Pour afficher les PBO à proximité de votre position sur la carte - **Quand** : Uniquement lorsque l'application est active et avec votre permission explicite ### 2. Données techniques - **Ce que nous collectons** : - Modèle de l'appareil - Version d'Android - Identifiant unique de l'appareil - **Pourquoi** : Pour améliorer les performances et corriger les bugs - **Quand** : Automatiquement lors de l'utilisation ### 3. Données d'utilisation - **Ce que nous collectons** : - Fonctionnalités utilisées - Fréquence d'utilisation - Rapports de crash - **Pourquoi** : Pour améliorer l'expérience utilisateur - **Quand** : Pendant votre utilisation de l'application ## Comment nous utilisons vos informations Nous utilisons vos informations pour : - Afficher les PBO proches de votre position - Améliorer les performances de l'application - Corriger les bugs et problèmes techniques - Développer de nouvelles fonctionnalités ## Partage de vos informations Nous ne vendons, n'échangeons ni ne louons vos informations personnelles à des tiers. Vos données peuvent être partagées uniquement dans les cas suivants : - Avec votre consentement explicite - Pour se conformer à une obligation légale - Pour protéger nos droits et notre sécurité ## Stockage et sécurité des données ### Stockage local - Les données des PBO sont stockées localement sur votre appareil - Aucune donnée personnelle n'est envoyée à nos serveurs sans votre consentement ### Sécurité - Nous utilisons des mesures de sécurité standard pour protéger vos données - L'accès aux données est limité aux personnes autorisées ## Vos droits Vous avez le droit de : - Accéder à vos données personnelles - Corriger vos données inexactes - Supprimer vos données - Refuser la collecte de données de localisation - Retirer votre consentement à tout moment ## Données des enfants PBO Manager n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles d'enfants de moins de 13 ans. ## Modifications de cette politique Nous pouvons mettre à jour cette politique de confidentialité. Nous vous informerons de tout changement en publiant la nouvelle politique dans l'application. ## Contact Pour toute question concernant cette politique de confidentialité, contactez-nous : **Email** : support@pbomanager.com **Adresse** : Ouagadougou, Burkina Faso ## Consentement En utilisant PBO Manager, vous acceptez cette politique de confidentialité. --- © 2025 PBO Manager.
